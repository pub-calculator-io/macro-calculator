Discover your ideal daily macronutrient intake with our free Macro Calculator. Get personalized protein, carb, and fat ratios to hit your fitness goals faster!

[https://www.calculator.io/macro-calculator/](https://www.calculator.io/macro-calculator/)

== Usage ==

Add the Macro Calculator shortcode to your page, post or sidebar:

`[calcio_macro_calculator]`

Add the following code to your template where you would like the Macro Calculator to appear:

`<?php display_calcio_macro_calculator(); ?>`

== Installation ==

1. Upload the Macro Calculator /calcio_macro_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Macro Calculator plugin through the "Plugins" menu in WordPress.
